11  * Platform-specific configuration.

13  * RADIOLIB_PIN_TYPE - which type should be used for pin numbers in functions like digitalRead().
14  * RADIOLIB_PIN_MODE - which type should be used for pin modes in functions like pinMode().
15  * RADIOLIB_PIN_STATUS - which type should be used for pin values in functions like digitalWrite().
16  * RADIOLIB_NC - alias for unused pin, usually the largest possible value of RADIOLIB_PIN_TYPE.
17  * RADIOLIB_SOFTWARE_SERIAL_UNSUPPORTED - defined if the specific platform does not support SoftwareSerial.
18  * RADIOLIB_HARDWARE_SERIAL_PORT - which hardware serial port should be used on platform that do not have SoftwareSerial support.
19  *
20  * In addition, some platforms may require RadioLib to disable specific drivers (such as ESP8266).
